# B.E.R.G. - CI Design System

## Farben

| Variable | Wert | Verwendung |
|----------|------|------------|
| `--primary` | `#7fd13b` | CTA, Highlights, Akzente |
| `--primary-hover` | `#6ab82e` | Hover-Zustande |
| `--primary-glow` | `rgba(127,209,59,0.15)` | Glow-Effekte |
| `--bg` | `#080808` | Seitenhintergrund |
| `--surface` | `#111111` | Karten, Panels |
| `--surface2` | `#1a1a1a` | Sekundare Flachen |
| `--border` | `rgba(255,255,255,0.08)` | Rahmen |
| `--text` | `#ffffff` | Primarer Text |
| `--text2` | `#aaaaaa` | Sekundarer Text |
| `--muted` | `#555555` | Dezenter Text |

## Fonts

| Variable | Font | Verwendung |
|----------|------|------------|
| `--font-display` | Barlow Condensed 900 | Headlines |
| `--font-body` | Barlow Condensed 700 | Subheadlines, Labels |
| `--font-text` | Barlow 400/600 | Fliesstext |

## Layout

| Variable | Wert |
|----------|------|
| `--content-max` | 1200px |
| `--radius` | 16px (Cards), 999px (Buttons/Chips) |

## Asthetik

- Military / Tactical Dark - Ton-in-Ton Schwarz, kein echtes Schwarz
- Neon-Grun (#7fd13b) als einziger Farbakzent
- Scan-Lines, Glitch-Effekte, Beam-Animationen
- Referenz: Call of Duty HUD, Night Vision Display
